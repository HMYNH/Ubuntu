Linux中有许多系统变量

系统变量PATH中 使用":"隔开了若干个路径
这些路径下的可执行文件,能够被认为是系统指令

# echo $PATH

如果用户自己编写的或通过其他方式获取的可执行文件 想要被当作指令使用
1.将 可执行文件 放入PATH识别的目录中
2.将 可执行文件 所在的路径添加至PATH
	<1>临时配置添加 export PATH=$PATH:/home/meson/Music
	   只对当前终端有效
	<2>永久配置 需要将 配置指令字段 写入"用户的环境配置文件" ~/.bashrc
	   "用户的环境配置文件" ~/.bashrc 经过修改后 不会立即生效
	   该文件在每次终端打开 确认用户身份时 会读取并配置
	   修改过环境配置文件后 需要在当前终端生效 使用指令 source 重新引入环境
	   # source ~/.bashrc
